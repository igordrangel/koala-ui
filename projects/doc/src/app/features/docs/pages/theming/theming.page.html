Theming

How to customize the appearance of your application using themes.

Koala UI uses the daisyUI library to provide a wide range of ready-to-use themes. You can easily apply these themes to your Angular application, ensuring a consistent look and feel throughout the user interface.

To apply a theme, choose one from the official list and apply it as shown below:

If you want to switch the theme by clicking or programmatically, simply change the value of the data-theme attribute.

To make the theme work, make sure to import the CSS file from ../node_modules/@koalarx/ui/theme/koala.css in your styles.css.

Note: It is worth mentioning that the koala.css file already imports Tailwind CSS and daisyUI.

Note: Line 2 is required to ensure that both Tailwind CSS and Daisy UI styles are also applied to Koala UI components.
