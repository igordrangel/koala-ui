Installation

How to install dependencies and structure your app.

Create project

Create a new project using the Angular CLI.

Install Koala UI

Install the Koala UI library in your project.

Import Koala CSS

Import the Koala CSS file in your styles.css to import the Koala Styles, Tailwind CSS and Daisy UI.

Note: Line 2 is required to ensure that both Tailwind CSS and Daisy UI styles are also applied to Koala UI components.

Add Tailwind CSS

Add Tailwind CSS plugin for PostCSS to a new .postcssrc.json file at root

Configure Koala Provider

Configure the Koala UI provider in your application.

Include App Root

Include the App Root component in your application.
